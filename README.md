# SVG_Logo_Maker

## Project Title:

SVG Logo Maker

## Description:

A Node.js command-line application that allows users to generate logos as SVG files by selecting colors, shapes, and text.

## Installation:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/hemuprabu/SVG_Logo_Maker

2. cd SVG_Logo_Maker

3. running npm install to install necessary packages.

## Usage:

To run the SVG Logo Maker application and generate logos, follow these steps:

1. Start the application by running the following command in your terminal:
   ``` bash
   node index.js
   ```

Follow the prompts to select a color, shape, and provide text for your logo.

- Choose the color by entering the corresponding number.

- Select the shape by entering the corresponding number.

- Enter the text you want to display on the logo when prompted.

- Once you have provided all the necessary input, the application will generate the logo and save it as an SVG file in the project directory.

- Locate the generated SVG file in the project directory and open it in a browser or SVG viewer to view your logo.

You can run the application multiple times to create different logos with varying colors, shapes, and text.

Enjoy creating logos with the SVG Logo Maker application!

## Features:

The SVG Logo Maker application offers the following key features for generating custom logos:

- **Color Selection**: Choose text and shape colors using color keywords or hexadecimal numbers.
- **Shape Selection**: Select from a variety of shapes including circle, triangle, and square for your logo design.
- **Text Input**: Input up to three characters to be displayed as text on the logo.
- **SVG File Generation**: Generate an SVG file based on your color and shape selections, along with the provided text, and save it as `logo.svg` for viewing and use.

Create unique logos with the SVG Logo Maker application by customizing colors, shapes, and text to suit your branding needs.
